# Proyecto de Comunicación Serial entre Java y Arduino

## Descripción
Este proyecto permite controlar LEDs conectados a un Arduino mediante una interfaz Java utilizando comunicación serial.

## Instrucciones de Configuración
1. Conecta los LEDs a los pines especificados en el código de Arduino.
2. Ejecuta la aplicación Java para controlar los LEDs.

## Estructura del Proyecto
- `/Proyecto_arduino/Proyecto_arduino.ino`: Contiene el código de Arduino.
- `Comunicacion_Serial/SerialCommunication/src/SerialCommunication.java`: Contiene el código Java.

## Requisitos
- Arduino IDE
- Java SDK

## Conexiones
- LED 1: Pin 2
- LED 2: Pin 3
- LED 3: Pin 4
- LED 4: Pin 5
